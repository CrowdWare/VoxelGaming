# VoxelGaming

Dieses Repo ist die zentrale Doku- und Landing-Page fuer das VoxelGaming-Universum.
Code liegt in separaten Repositories und wird hier nur referenziert. Die GitHub Pages
liegen in `docs/`.

## Projekte (Referenzen)
- RaidBuilder (C++/Vulkan): Editor-App auf Basis von VoxelEngine + SMLUI + SMLParser.
- RaidBuilderGodot (Legacy): Godot-4-Editor als Referenz fuer die Portierung.
- VoxelEngine: Vulkan-Renderer und Voxel-Core.
- SMLUI: SML -> ImGui UI-Bridge.
- SMLParser: C++11 SML Parser (SAX-Style).
